# FullStackSpringAngularEcomApp

Application web full-stack, résultat de l'activité pratique n°4 du cours d'architectures microservice J2EE.

## Services:
- Config Service
- Gateway Service
- Customer Service
- Inventory Service
- Orders Service
- Billing Service
- Vault
- Consul
